# 🌙 ZwerBo – Ultimate Edition  
**Version 3.0.0 – Ausgewogen Plus**

ZwerBo ist ein magischer, humorvoller und warmherziger Discord‑Begleiter, der auf Nachrichten reagiert, Elemente zieht, Runen deutet und mit einer eigenen Persönlichkeit antwortet.  
Die Ultimate Edition vereint eine verbesserte Struktur, ein modernes Trigger‑System und neue magische Features – ohne XP‑System.

---

## ✨ Features

### 🔮 Persönlichkeitsmodul
- Mystisch, warm, humorvoll und verspielt  
- Dynamischer Tonfall je nach Trigger  
- Konsistente Charakterstimme

### 🌙 Trigger‑System
- Synonyme  
- Fuzzy‑Erkennung  
- Kontext‑Erkennung  
- Prioritäten  
- Erweiterbar über Tabellen

### 🔥 Magische Fähigkeiten
- `/element` → zieht ein zufälliges Element  
- `/rune` → zieht eine Rune + Bedeutung  
- `/version` → zeigt die aktuelle Version  
- Mini‑Lore, Gedanken, magische Antworten

### 🧩 Struktur
- Modularer Aufbau  
- Leicht erweiterbar  
- Sauber getrennte Module  
- Wartungsfreundlich

---

## 🛠 Installation

1. Repository klonen oder herunterladen  
2. `main.py` und `requirements.txt` prüfen  
3. In Render oder lokal starten  
4. Environment Variable `TOKEN` setzen  
5. Bot starten
